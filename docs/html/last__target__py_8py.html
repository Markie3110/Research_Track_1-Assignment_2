Detailed Description

A rosnode that monitors the last position specified by the user.

Author
Mark Henry Dsouza
Date
15/05/2024

Description:
Implements the ROS node "last_target" that creates the rosservice "last_target_serv" which when called, prints the latest goal position specified by the user. Obtains the latest goal via a subscription to the "/goal" topic of the "reaching_goal" actionserver.

Subscribes to:

  • /reaching_goal/goal

Publishes to:

  • [None]

Services implemented:
